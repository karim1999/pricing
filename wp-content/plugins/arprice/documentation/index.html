Installation And Activation

You can install plugin either of these two methods.

New install via FTP

  1. Download the plugin from CodeCanyon, and extract it. You should find a directory arprice.
  2. In your WordPress installation, upload arprice to wp-content/plugins/
  3. Activate the "ARPrice" plugin through the Plugins menu from the WordPress admin panel.

New install via the WordPress Uploader

  1. Download the plugin from CodeCanyon
  2. Log in to your WordPress admin panel
  3. Go to Plugins -> Add New -> Upload
  4. Click on choose file and browse the download package from CodeCanyon. When the plugin uploads successfully, click on Activate Plugin.

After activating the plugin, you will be able to see the ARPrice menu in the admin left menu.

ARPrice License Activation

After activating the plugin, you need to activate your license for support and automatic upgrades. You will get the pricing table analytics feature only after activating your license. A license key will only be used for one installation of WordPress at a time.

Click on Settings link under "ARPrice" Menu to activate your license. You will see the below section there on that page. To activate your license, you need to enter your Name, Email address and purchase code( Unique code obtained while purchased this product).

ARPrice Update

Auto Update

Please, take a backup of your database, your all pricing tables, uploads > arprice folder and Plugins > arprice folder

  1. In your WordPress admin, go to Plugins > Installed Plugins
  2. m here you can update ARPrice Plugin directly from an available update link for ARPrice.

Manual Update

  1. Download the latest version of the plugin.
  2. Unzip the downloaded file to your computer.
  3. Take Backup of ARPrice directory (backup must be kept outside of plugins folder otherwise it will give an error).
  4. Upload the new plugin source code /ARPrice/ directory to the /wp-content/plugins/ directory of your site by using FTP software or using File Manager in the hosting panel.
  5. Activate the plugin through the 'Plugins' menu in WordPress

Chargebee Integration

ARPrice supports Chargebee Integration. You need to follow the below steps to integrate Chargebee with ARPrice.

  1. You'll have a script tag received from the Chargebee account which you need to place in the header section of your website. To do so, you can use the following methods.

    2. Using theme header file (Recommended)

    • You can add the script tag in your theme header.php file.
    • You can modify the header.php using the WordPress theme editor or you can use any editor plugin to modify the file.
    • The script tag will look like this: <script src="https://js.chargebee.com/v2/chargebee.js" data-cb-site="your site name"></script>
    • Add the above script tag before </head>.
    • Save the file and that's it.
    • Note: If you update the theme, then you need to follow the above steps again as updating the theme will remove the code.

    Using third party plugin

    • You can use any third-party plugin which gives the facility to add a script into the header
    • If you want to use this method, we suggest (but not recommended as third party plugins are not a part of ARPrice and may cause conflict so please use the plugin which works on your website) Insert Headers and Footers plugin.
    • Go to Dashboard > Settings > Insert Headers and Footers page
    • Insert <script src="https://js.chargebee.com/v2/chargebee.js" data-cb-site="your site name"></script> code in the Scripts in Header section and save the settings.
  2. Now, you'll need to add the button in the ARPrice. Please follow the below steps for that.
    • Open the pricing table in the editor
    • Open the pricing table button link settings
    • Add your button code with an extra class attribute to match the Chargebee button design with the pricing table. Add <a href="javascript:void(0)" data-cb-type="checkout" class="bestPlanButton" data-cb-plan-id="Chargebee Plan Id">Your Button Text</a> in the embed script and enable Hide default button option.

    • Save the pricing table and that's it.

A/B Testing

A/B testing (also known as split testing or bucket testing) is a method of comparing two versions of a webpage or app against each other to determine which one performs better. AB testing is essentially an experiment where two or more variants of a page are shown to users at random, and statistical analysis is used to determine which variation performs better for a given conversion goal.

ARPrice provides the A/B testing facility to display more than one pricing table using just single Shortcode. You need to add shortcode : [ARPriceABTest] in frontend after configuring pricing table priority from admin panel.

Higher the percentage of table, it will display more times in frontend.
